# Web Corporativa de [FoodLogistics]

Aquesta és la web corporativa de Food Logisitcs S.A, una empresa fictícia dedicada principalment a la comercialització i distribució a l’engròs de productes alimentaris, operant dins del sector logístic i de subministrament d’aliments, dins el mòdul *Projecte Intermodular*.

## Demostració

[🌐 URL de la web](https://polabenza.github.io/web_corporativa_pac/)

## Tecnologies Utilitzades

Per la creació d'aquesta web, s'han utilitzat les següents tecnologies i eines:

- Claude/Gemini
- HTML5/CSS3
- Statcounter per l'analítica
